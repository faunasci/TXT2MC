# TXT2MC - Browser-Based Markov Chain Trainer

[(Português)](#português) | [(Français)](#français)

## English

### Description
**TXT2MC** is a lightweight, client-side Markov Chain text generator. It allows you to train a statistical model directly in your browser using local `.txt` files. No server, no APIs, and no data leaves your machine.

It's a fast statistical approximation of how "Large Language Models" process sequences, using word-level N-Grams to predict the next word in a sequence.

### Features
- **Zero Dependencies**: Pure HTML/CSS/JavaScript.
- **Privacy First**: All processing is done locally via File API.
- **Configurable Context**: Custom N-Gram size (1-5) to control coherence.
- **Interactive Playground**: Real-time text generation based on trained data.

### How to Use
1. Open `index.html` in any modern web browser.
2. Drag and drop or select a `.txt` file for training.
3. Choose the N-Gram size (2 is recommended for small/medium texts).
4. Click **Train Model**.
5. Type a prompt or leave it blank and click **Generate Text**.

## Français

### Description
**TXT2MC** est un générateur de texte basé sur les chaînes de Markov, fonctionnant entièrement côté client dans le navigateur. Il vous permet d'entraîner um modèle statistique à partir de fichiers `.txt` locaux. Aucun serveur, aucune API, et aucune donnée ne quitte votre machine.

Il s'agit d'une approximation statistique simple de la manière dont les modèles de langage traitent les séquences, utilisant des N-Grams au niveau des mots pour prédire le mot suivant.

### Caractéristiques
- **Zéro Dépendance** : HTML/CSS/JavaScript pur.
- **Confidentialité Totale** : Tout le traitement est effectué localmente via File API.
- **Contexte Configurable** : Taille de N-Gram personnalisable (1-5) pour contrôler la cohérence.
- **Aire de Jeux Interactive** : Génération de texte en temps réel basée sur les données d'entraînement.

### Comment l'utiliser
1. Ouvrez `index.html` dans n'importe quel navigateur web moderne.
2. Glissez-déposez ou sélectionnez un fichier `.txt` pour l'entraînement.
3. Choisissez la taille du N-Gram (2 est recommandé pour les textes courts/moyens).
4. Cliquez sur **Train Model**.
5. Saisissez un prompt ou laissez vide et cliquez sur **Generate Text**.
